token_generator(tknsFile)

source code 

Create an iterator object for each token of the tokens file.

The elements of the iterator are tokens of the file as strings. It looks like: sidelines

Parameters:
  • tknsFile (FILE) - The file containing the tokens.
Returns: generator
An iterator generating tokens as strings.

Note: Iterator elements representing empty lines are None.

Warning: The input file must contain one token per line.

viterbi(sentence)

source code 

Viterbi alorithm for finding the mst likely tag for every tokens.

The Viterbi algorithm is a dynamic programming algorithm for finding the most likely sequence of hidden states – called the Viterbi path – that results in a sequence of observed events, especially in the context of Markov information sources and hidden Markov models.

The function has been implemented based on the following algorithm: Input: ====== Z=z1,z2,...,zn the input observed sequence

Initialization:

k=1 time index S(c1)=c1 L(c1)=0 variable accumulating the lengths, the initial length is 0

Recursion:

For all transitions tk=(ck,ck+1) ....compute: L(ck,ck+1) = L(ck) + l [tk = (ck,ck+1)] among all ck. Find L(ck+1) = min L(ck,ck+1) For each ck+1 ....store L(ck+1) and the corresponding survivor S(ck+1). k=k+1 Repeat until k=n

Parameters:
  • sentence (list) - A list of tokens (strings) representing a sentence.
