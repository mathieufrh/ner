token_generator(tknsFile)

source code 

Create an iterator object for each token of the tokens file.

The elements of the iterator are tuples containing a token and its associated tag. It looks like: (David, I-PER)

Parameters:
  • tknsFile (FILE) - The file containing the tokens.
Returns: generator
An iterator generating tuples of tokens and token tags.

Note: Iterator elements representing empty lines are tuple with token and associated tag set to None.

Warning: The input file must contain one token per line.

sentence_generator(toknIterator)

source code 

Create an iterator object for each sentence of the generated tokens.

The elements of the iterator are lists of tuples. Every tuples are containing a token and its associated tag. It looks like: [('3.', 'O'), ('Goulnara', 'I-PER'), ('Fatkoullina', 'I-PER'), ('Russia', 'I-LOC')]

Parameters:
  • toknIterator (generator) - A generator iterating on each token of a file.
Returns: generator
An iterator generating lists of tuples of tokens and token tags.

ngram_generator(sntncIterator, n)

source code 

Create an iterator object for each n-gram generated from tokens lists.

The elements of the iterator are tuples of tuples. Every sub-tuples are containing a token and its associated tag. It looks like: (('Division', 'O'), ('three', 'O'), (None, 'STOP'))

The sentence boundaries are added by inserting '*' and 'STOP' tokens. The elements of the generated iterator are all of length n.

Parameters:
  • sntncIterator (generator) - A generator iterating on each sentence of a file.
Returns: generator
An iterator generating n-grams represented by tuple of n tuples including sentence boundaries tokens.
