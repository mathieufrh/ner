token_generator(tknsFile, addProb=False)

source code 

Create an iterator object for each token of the tokens file.

The elements of the iterator are tuples containing a token and its associated tag. It looks like: (David, I-PER)

Parameters:
  • tknsFile (FILE) - The file containing the tokens.
Returns: generator
An iterator generating tuples of tokens and token tags.

Note: Iterator elements representing empty lines are tuple with token and associated tag set to None.

Warning: The input file must contain one token per line.
